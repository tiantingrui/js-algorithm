# 数组

单纯针对数组来考察的题目，总体来说都不算太难。

一般都会结合排序、二分和动态规划这些相对复杂的算法思想综合考察。



#### 几个数组必知必会的代码实现

+ 实现一个支持动态扩容的数组
+ 实现一个大小固定的有序数组，支持动态增删改操作
+ 实现两个有序数组合并为一个有序数组
